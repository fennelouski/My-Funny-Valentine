# Prompt 09: Marketing Website

## Objective
Create a static marketing website to promote the app, showcase features, and drive App Store downloads.

## Context
The website is a marketing tool deployed on Vercel. It should be modern, responsive, and optimized for conversions.

## Reference Documentation
- `docs/11-WEBSITE-MARKETING.md` - Complete website requirements
- `docs/01-FEATURE-OVERVIEW.md` - App features

## Tasks

### 1. Set Up Website Project
- Initialize Next.js project (or plain HTML/CSS/JS)
- Configure for static export
- Set up Vercel deployment
- Configure build settings
- Set up project structure

### 2. Create Landing Page
- Hero section with app name and tagline
- Key features showcase (5 main features)
- Screenshots gallery
- App Store download buttons
- Call-to-action sections
- Footer with links

### 3. Design Hero Section
- Headline: "Create Personalized Valentine's Cards with AI"
- Subheadline with value proposition
- Primary CTA: "Download on the App Store"
- App icon or hero image
- Responsive design

### 4. Create Features Section
- Feature 1: AI-Generated Sayings (with visual)
- Feature 2: Face Detection (with visual)
- Feature 3: Image Integration (with visual)
- Feature 4: iCloud Sync (with visual)
- Feature 5: Social Sharing (with visual)
- Each with description and visual

### 5. Create Screenshots Gallery
- Screenshot 1: Card creation interface
- Screenshot 2: AI sayings generation
- Screenshot 3: Face import workflow
- Screenshot 4: Card gallery
- Screenshot 5: Sharing options
- Lightbox or carousel display

### 6. Implement App Store Integration
- App Store download badge
- Direct App Store link
- QR code for mobile access
- Multiple badge sizes
- Localized badges (if needed)

### 7. Add Pricing Information
- Free tier description
- Premium subscription details
- Value proposition
- Clear pricing display
- FAQ section (optional)

### 8. Optimize for SEO
- Meta tags (title, description)
- Open Graph tags
- Structured data (JSON-LD)
- Sitemap.xml
- Robots.txt
- Semantic HTML

### 9. Implement Analytics
- Vercel Analytics integration
- Google Analytics (optional)
- Track App Store link clicks
- Track page views
- Conversion tracking

### 10. Optimize Performance
- Image optimization (WebP format)
- Lazy loading images
- Code splitting
- Minify CSS/JS
- Lighthouse score >90
- Fast page load (<2 seconds)

## Deliverables
- Complete marketing website
- Responsive design (mobile, tablet, desktop)
- SEO optimized
- Analytics integrated
- Performance optimized
- App Store links working
- Deployed on Vercel

## Notes
- Use Valentine's theme colors (reds, pinks)
- Modern, clean design
- Mobile-first approach
- Fast loading times
- Clear call-to-actions
- Professional appearance
